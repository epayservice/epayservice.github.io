OpenAPI permite a los proveedores de pagos y desarrolladores externos (TPP) obtener acceso a nuestras tecnologías, productos y servicios para desarrollar nuevas herramientas y lanzar soluciones innovadoras al mercado sin perjudicar a los clientes.

¿Cuál es la ventaja de OpenAPI?

OpenAPI permite a los bancos y servicios financieros usar sus recursos de forma segura. Gracias a la apertura al mundo exterior, OpenAPI ayuda a encontrar las mejores y más efectivas soluciones.
  • Seguridad mejorada
  • El cliente posee sus datos
  • Integración fácil
  • Responsabilidad

Protección de Datos

Antes de proporcionar acceso a los datos del cliente, requerimos su aprobación. Todos los datos están protegidos y se requiere autorización para acceder a ellos a través de ePayService OpenAPI. Para autenticar a los usuarios, utilizamos estándares de seguridad como el protocolo OAuth2 y el sistema OpenID Connect.
